# Module: Coding Basics (for kids)

## 1. Purpose

Introduce primary-school kids to programming through visual tools (Scratch,
micro:bit) in a friendly, no-pressure setting. Bridges the gap for families
where coding isn't part of the home environment.

## 2. Target group

- **Age band(s)**: `7-12`
- **Participant profile**: grades 1–6, all skill levels
- **Barrier considerations**: free, snack provided, no prior experience needed

## 3. Activity patterns

- **Frequency**: weekly
- **Duration**: 60 minutes
- **Format**: guided workshop with 3 rotating stations
- **Facilitators**: 2 volunteers + 1 parent assistant per 8 kids
- **Equipment / space**: 6 tablets or laptops, 4 micro:bit kits, tables

## 5. ROI drivers

- **Cost per participant**: target 500–800 SEK / participant / month
- **Grant leverage ratio**: target 1.5
- **Continuity**: 9 months / year (school term aligned)
- **Multiplier**: kids graduate to Youth Tech at age 13

## 6. Grant relevance

- **Kommunala bidrag** — early digital literacy programs
- **Arvsfonden** — family inclusion and skills

**Evidence required**: weekly counts, age breakdown, session plans.

## 7. Yearly reusability plan

- **Year 1**: Scratch basics, 4 sessions per term
- **Year 2**: add micro:bit + physical computing
- **Year 3**: introduce a "family coding day" for parents + kids
- **Retirement criteria**: < 6 participants for 2 consecutive months
